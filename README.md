# Nas Dashboard

A beautiful and minimalistic dashboard for your NAS. Static as possible.

## Demo

[GitHub Pages](https://cworld1.github.io/nas-dashboard/)

## Usage

1. Clone the repository.
2. Rename/copy `.example.services.json` to `services.json` and edit it to fit your needs.
3. Open `index.html` in a web browser for a test. If you have Python, you can run `python -m http.server` in the project directory to serve the files.
4. After testing, you can deploy the dashboard to your NAS or any web server.
